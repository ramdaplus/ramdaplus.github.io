_R.sum(array, [key])
Adds up all the elements of the array
Since: 0.1.0
Arguments
array (Array): The array whose sum to be calculated
key (String): Key of objects whose sum needs to be calculated. In case of array of objects
Returns
(Number/Array): Returns a sum/array of sum if array of object is passed
Example
const array = [ 1,2,3,4 ];
_R.sum(array);
// => 10

const arrayOfObjects = [ { a: 1, b: 3 }, { a: 2, b: 4 } ];
_R.sum(arrayOfObjects, [ a, b ]);
// => [ 3, 7 ]
